AZ Accordion

Default settings

var azAccordion = new AZAccordion(
{
    azAccordionId: "",
    azAccordionHeightStyle: "content",
    azAccordionCollapsible: true,
    azAccordionOpenEvent: "click",
    azAccordionHeaderBackgroundColor: "#009688",
    azAccordionHeaderColor: "#FFFFFF",
    azAccordionHeaderHoverBackgroundColor: "#607D8B",
    azAccordionHeaderHoverColor: "#FFFFFF",
    azAccordionArticleBackgroundColor: "#FFFFFF",
    azAccordionArticleColor: "#000000",
    azAccordionIconClosed: "fas fa-plus",
    azAccordionIconOpen: "fas fa-minus",
    azAccordionSlideDown: 100,
    azAccordionSlideUp: 100
});

Tilleggsfunksjoner

Hvis du ønsker å bruke egne knapper eller linker til å åpne/lukke AZAccordion, bruker du funksjonen under.

azAccordion.SelectAccordion(0);


Du har også muligheten til å skifte ut innholdet i AZAccordion, ved enten å klikke på en knapp eller link, eller en bestemt hendelse inntrer.

azAccordion.ChangeText(0, "Dette er en tekst");

Det første parameteret, (Index) tilsvarer nummert på hvilken accordion "header" du ønsker å gjøre noe med. Det siste parameteret er selve innholdet.

azAccordionOpenEvent Type: String Default: "click" Values: "click" or "mouseover"
azAccordionHeaderBackgroundColor Type: Color Hex Default: "009688"

$.subscribe("functionlib/azAccordionHeader", function (e, data)
{
    data.azAccordionId
    data.azAccordionStatus
    data.azAccordionActivated
    data.azAccordionDeactivated
    data.azAccordionHeaderJQElement
});

Denne funksjonen trigges hver gang man klikker, (åper/lukker) din azAccordion. Ved å abonnere på funksjonen, vil du få en bedre flyt i din egen kode.
