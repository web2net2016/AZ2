AZ Accordion

Default settings

new AZAccordion(
{
    azAccordionId: "",
    azAccordionHeightStyle: "content",
    azAccordionCollapsible: true,
    azAccordionOpenEvent: "click",
    azAccordionHeaderBackgroundColor: "#0078D7",
    azAccordionHeaderColor: "#FFFFFF",
    azAccordionHeaderHoverBackgroundColor: "#0bb2d4",
    azAccordionHeaderHoverColor: "#FFFFFF",
    azAccordionArticleBackgroundColor: "#FFFFFF",
    azAccordionArticleColor: "#000000",
    azAccordionIconClosed: "fas fa-plus",
    azAccordionIconOpen: "fas fa-minus",
    azAccordionSlideDown: 100,
    azAccordionSlideUp: 100
});

Additional Features

If you want to use your own buttons or links to open/close the AZAccordion, use the function below.

azSelectAccordion(0);


You also have the option to replace the contents of the AZAccordion, by using the function below.

azChangeText(0, "This is a text");

The first parameter, (Index) corresponds to the number of which AZAccordion "header" you want to interact with. The last parameter is the content itself.




azAccordionId Type: String Default: ""
azAccordionHeightStyle Type: String Default: "content" Values: "content" or "auto"
azAccordionCollapsible Type: Boolean Default: true Values: true or false
azAccordionOpenEvent Type: String Default: "click" Values: "click" or "mouseover"
azAccordionHeaderBackgroundColor Type: Color Hex Default: "0078D7"
See also CSS .az-accordion
azAccordionHeaderColor Type: Color Hex Default: "FFFFFF"
See also CSS .az-accordion
azAccordionHeaderHoverBackgroundColor Type: Color Hex Default: "607D8B"
See also CSS .az-accordion
azAccordionHeaderHoverColor Type: Color Hex Default: "FFFFFF"
See also CSS .az-accordion
azAccordionArticleBackgroundColor Type: Color Hex Default: "FFFFFF"
See also CSS .az-accordion
azAccordionArticleColor Type: Color Hex Default: "000000"
See also CSS .az-accordion
azAccordionIconClosed Type: String Default: "fas fa-plus"
azAccordionIconOpen Type: String Default: "fas fa-minus"
azAccordionSlideDown Type: Integer Default: 100 Unit: milliseconds
azAccordionSlideUp Type: Integer Default: 100 Unit: milliseconds
functionlib/azAccordionHeader Type: Function
$.subscribe("functionlib/azAccordionHeader", function (e, data)
{
    data.azAccordionId
    data.azAccordionStatus
    data.azAccordionActivated
    data.azAccordionDeactivated
    data.azAccordionHeaderJQElement
});

This feature is triggered every time you click, (open/close) the AZAccordion. By subscribing to the feature, you will get a better flow in your own code.
